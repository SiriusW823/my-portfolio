# SIRIUS — Personal Portfolio

A terminal-inspired personal portfolio focused on security, quantum computing, engineering, competitions, credentials, projects, and writing.

## Live site

https://siriusw823.github.io/my-portfolio/

## Highlights

- Custom terminal-inspired visual identity
- Introduction and personal focus areas
- Competitions and results section
- Credentials and certifications
- Project highlights
- Writing archive and About page
- Responsive navigation
- Language control
- Skip-to-content accessibility link

## Technology

The site is deployed with GitHub Pages from this public repository. See the source files in the repository for the frameworks and tools used by the current implementation.

## Running locally

Clone the repository, install the dependencies listed by the project, and run the development command defined in `package.json`.

```bash
git clone https://github.com/SiriusW823/my-portfolio.git
cd my-portfolio
npm install
npm run dev
```
